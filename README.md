# alphacricket

## Setup

1. Create a virtual environment:

   ```
   python3 -m venv alphacricket_env
   ```

2. Activate the virtual environment:

   ```
   source alphacricket_env/bin/activate
   ```

3. Install dependencies:

   ```
   pip install -r requirements.txt
   ```

4. To run:

   ```
   python3 src/main.py
   ```
